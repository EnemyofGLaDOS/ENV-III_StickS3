# ENV-III_StickS3
A real-time environmental monitoring firmware for the M5StickS3, designed to work with the M5Stack ENV-III Unit.

This firmware reads temperature, humidity, and barometric pressure and displays the data on-screen in a clean, readable layout optimized for quick reference.

Built as part of a modular firmware workflow, this project serves as a core environmental sensing module that can be expanded into larger systems such as weather logging, GPS-based data tagging, or multi-sensor dashboards.

🌡️ Real-time Temperature monitoring (°F display)
💧 Real-time Humidity monitoring (% RH)
🌬️ Barometric Pressure readings (hPa)
📺 Clean multi-panel display layout
🔄 Continuous live updates
🧱 Designed for modular expansion
⚡ Fast boot and stable sensor polling
🧠 Efficient memory usage for ESP32-S3 devices
